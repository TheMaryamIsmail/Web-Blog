Introduction to Web:

The Web (World Wide Web) is a system of interlinked hypertext documents and multimedia content, such as text, images, videos, and other files, that are accessed through the Internet. It allows users to browse and interact with content hosted on servers worldwide using web browsers.
The Web is often confused with the Internet, but they are different:

  • The Internet is the global network of interconnected computers and servers that enables communication between devices.
  • The Web is a service that operates over the Internet, using the HTTP/HTTPS protocols to transfer data.

    • How The Web Works

    The Web Works as below;

    Web Browsers
    To access the Web, you use a web browser (like Chrome, Firefox, Safari, or Edge). The browser allows you to request and view web pages.
    Web Servers
    Websites and web pages are hosted on web servers, which store the files and data that make up the site. When you request a page by typing in a URL, the web server sends the page's data to your browser
    Url(Universal Resource Locator)
    A URL is the address of a web page. It tells the browser where to find the web content on the server. For example, https://www.example.com.
    HTTP/HTTPS(Hypertext Transfer Protocol)
    These are protocols used for transferring web pages over the Internet. HTTP is the basic protocol, while HTTPS is a secure version that encrypts the data.
    DNS(Domain Name System)
    When you type a URL into the browser, the DNS system translates that domain name (like www.example.com) into an IP address that the server can recognize.
    Rendering
    Once your browser receives the data from the web server, it interprets the HTML, CSS, and JavaScript code to render the web page on your screen in a readable and interactive format.
